College Webpage

We have built a website for a college as part of the project assigned for the Web Development Subject.

Railway Reservations

We have built a website for railway reservations as part of the project assigned for the Database Management System Subject.

This project focused on developing a comprehensive website for railway reservations, providing users with a streamlined booking experience. Key features of the application included:

  • User authentication and secure login system
  • Search functionality for trains based on routes, timings, and availability
  • Booking management with seat selection and payment integration
  • Admin panel for managing train schedules, seat availability, and user bookings
  • Interactive interface with responsive design for optimal viewing on various devices

The front-end was crafted using HTML, CSS, and JavaScript to ensure a visually appealing and intuitive user experience. PHP was employed for server-side scripting, while MySQL facilitated robust database management for storing and retrieving booking information securely.
